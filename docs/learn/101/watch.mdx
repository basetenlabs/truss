In this step, we'll patch our deployed model server with this new model.

**Deployment**

```sh
truss watch
```

**Invocation**

```sh
truss predict "I am happy"
```

**Response**

```
POSITIVE
```

Our model server is running in production as a draft, so we can patch it to continue developing our model.

<RequestExample>

```python model/model.py
from typing import Any
from transformers import pipeline


class Model:
    def __init__(self, **kwargs) -> None:
        self._data_dir = kwargs["data_dir"]
        self._config = kwargs["config"]
        self._secrets = kwargs["secrets"]
        self._model = None

    def load(self):
        self._model = pipeline("text-classification")

    def predict(self, model_input: Any) -> Any:
        return self._model(model_input)
```
