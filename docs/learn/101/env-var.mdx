title: "Step 8: Set environment variables"
description: "Description"
---

In this step, we'll add an environment variable.

This is a bit contrived, the model here doesn't really need one. But it shows off `truss watch`.

<Tabs>
<Tab title="Code">
```yaml config.yaml
environment_variables:
  bar: bar_bar
```
</Tab>
<Tab title="Diff">
```diff config.yaml
-environment_variables: {}
+environment_variables:
+  bar: bar_bar
```
</Tab>
</Tabs>

<RequestExample>

```yaml config.yaml  â—
environment_variables:
  bar: bar_bar
external_package_dirs: []
model_metadata: {}
model_name: null
python_version: py39
requirements:
  - torch==2.0.1
  - transformers==4.30.0
resources:
  accelerator: null
  cpu: 500m
  memory: 512Mi
  use_gpu: false
secrets: {}
system_packages: []
```

```python model/model.py
from typing import Any
from transformers import pipeline


class Model:
    def __init__(self, **kwargs) -> None:
        self._data_dir = kwargs["data_dir"]
        self._config = kwargs["config"]
        self._secrets = kwargs["secrets"]
        self._model = None

    def load(self):
        self._model = pipeline("text-classification")

    def predict(self, model_input: Any) -> Any:
        return self._model(model_input)
```
