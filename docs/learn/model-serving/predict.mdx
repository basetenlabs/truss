In this step, we'll invoke our deployed "model" (aka echo server).

**Invocation**

```sh
truss predict --request '{"prompt": "Hello, World"}'
```

**Response**

```
{"prompt": "Hello, World"}
```

Our model server is up and running, and we can push to it, watch for changes to go live, and run model inference.

The next three steps will cover implementing a real model in your Truss.
